# Changelog Notes

This directory contains phase-specific notes and changelog entries for work done on the book plan.

## Naming Convention

Phase notes should be named using the format: `{chapter_number}-{phase_number}_notes.md`

Examples:
- `0-1_notes.md` - Chapter 0, Phase 1 (Content Creation)
- `1-2_notes.md` - Chapter 1, Phase 2 (Review)
- `3-1_notes.md` - Chapter 3, Phase 1 (Content Creation)

## Phase Numbers

- Phase 1: Content Creation
- Phase 2: Review
- Phase 3: Editorial

## Usage

Each phase notes file should contain:
- Important decisions made during the phase
- Blockers or issues encountered
- Observations and learnings
- Any context that would be helpful for future reference
